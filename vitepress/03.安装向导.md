# 安装与安装向导

## 在cmd窗口运行下面的命令

```
npm add -D vitepress
```

## 安装向导

VitePress 附带一个命令行设置向导，可以帮助你构建一个基本项目。安装后，通过运行以下命令启动向导：

```
npx vitepress init
```
